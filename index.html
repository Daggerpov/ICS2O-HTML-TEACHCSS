What is CSS?

Well, CSS stands for Cascading Style Sheets

It's used in tandam with HTML in order to describe various HTML elements. This means that it can adjust previously coded HTML.

It's useful to describe these elements in general, rather than individually having to describe them in the HTML tag as shown below:

CSS saves you time and energy, by essentially being like a function in the sense that, instead of having to repeat a process multiple times, you can simply call that function. In this case, this would describe how describing an HTML element in CSS can be applied to all instances of that element, rather than having to describe it in the HTML script for every single instance.

How CSS Works

Selector

The selector is a pointer to the HTML element you want to adjust with your rule-set. It's made up of one or more elements.

Declaration Blocks

The declarations are made up of the rules you want to use in order to adjust said elements. They are made up of properties and values, separated by a colon and ended off with a semicolon. You can find various properties by following this link.

Declaration blocks are surrounded by curly braces.

Internal CSS

An internal stylesheet holds CSS rules for the page in the 'head' section of the HTML file as shown below:

This can only be used to define a style for one singular HTML page, rather than being able to do it for multiple like External CSS can.

Other Selectors

These can both be selected in CSS code, just like an HTML element in a selector as shown below:

ID Selector

The ID selector uses the ID attribute of an HTML element to select a specific element, as opposed to all elements with the same tag like 'p'. The ID of an element is unique within a page. In order to select an element by its unique ID, type the hash character, followed by the element's ID.

Class Selector

Similarly to the ID selector, it selects HTML elements with a specific class attribute. However, this isn't unique to a specific element, since multiple elements can be objects with the same class attributes.
